<h1>Institute ERP Admin Panel (Frontend)</h1>
<p>A responsive Admin Panel built with React.js for managing institute operations including students, fees, library, study materials, and announcements.</p>

<h2>#Features</h2> <br>
<ul>
<li>Login Authentication (Role-Based) </li>
<li>Dashboard Overview </li>
<li>Student Management </li>
<li>Fee Collection System</li>
<li>Library Management </li>
<li>Study Material (PDF View & Download) </li>
<li>Announcement System </li>
<li>Document Upload & Tracking </li>
<li>Fully Responsive UI </li>
<li>API Integration </li>
</ul>
